## ğŸ¨ Customization

### Change Colors
Edit CSS variables in `index.html`:
```css
:root {
    --color-primary: #1a1a1a;
    --color-accent: #d4a574;
    --color-secondary: #4a5f4e;
    --color-light: #f5f3ef;
}
```

### Change Fonts
1. Pick fonts from https://fonts.google.com
2. Replace Google Fonts link in `<head>`
3. Update CSS variables:
```css
--font-display: 'YourFont', serif;
--font-body: 'YourFont', sans-serif;
```
